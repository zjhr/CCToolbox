# Coding-Tool 智能安装使用指南

## 📦 安装工具

```bash
npm install -g coding-tool
```

## 🚀 使用智能安装

### 1. 启动工具

```bash
ct
```

### 2. 选择智能安装

在菜单中选择 `🚀 智能安装`

### 3. 选择要配置的 CLI 类型

- **Claude Code** - Anthropic 官方 CLI
- **Codex** - OpenAI Codex CLI
- **Gemini** - Google Gemini CLI

### 4. 填写配置信息

#### 渠道名称
给配置起个名字，例如：`Claude Official`、`My Codex`

#### Provider 名称（仅 Codex）
给 Provider 起个名字，例如：`custom`、`myapi`
- 只能包含小写字母、数字、下划线和连字符
- 这个名称会用于 config.toml 和环境变量

#### Base URL
API 服务地址，例如：
- `https://api.anthropic.com`
- `https://api.openai.com`
- `https://your-proxy.com`（如果使用代理）

#### API Key
你的 API 密钥（输入时会自动隐藏）

### 5. 自动完成

工具会自动：
- ✅ 检测 CLI 是否已安装（未安装会提示安装）
- ✅ 生成配置文件
- ✅ 添加渠道到管理系统
- ✅ 验证配置是否成功

## ✨ 配置说明

### Claude Code
- 配置文件：`~/.claude/settings.json`
- 会保留你现有的 hooks 等配置

### Codex
- 配置文件：
  - `~/.codex/config.toml` - 添加 provider 配置
  - `~/.codex/.env` - 存储 API Key
- Provider 会被设置为默认 provider

### Gemini
- 配置文件：`~/.gemini/.env`
- 不会影响现有的 `settings.json`

## 💡 常见问题

**Q: 提示权限不足怎么办？**
A: 工具会自动询问是否使用 sudo 安装，选择 Yes 并输入密码即可

**Q: 原来的配置会被覆盖吗？**
A: 不会！工具会自动备份原配置

**Q: 安装失败怎么办？**
A: 根据提示手动安装：
```bash
npm install -g @anthropic-ai/claude-code
# 或
npm install -g codex-cli
# 或
npm install -g @google/generative-ai-cli
```

## 🎯 验证配置

安装完成后，测试一下：

```bash
# Claude Code
claude --version

# Codex
codex --version

# Gemini
gemini --version
```

## 🔧 管理渠道

安装完成后，可以通过 Web UI 管理所有渠道：

```bash
ct ui
```

浏览器会自动打开管理界面：`http://localhost:9999`
